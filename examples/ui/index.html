What is Dalila UI? Dalila UI is a CSS-only design system built with warm neutrals, accent colors, and smooth transitions. It provides a collection of accessible, composable components. Is it accessible? Yes! All components are built with semantic HTML and follow WAI-ARIA patterns. Keyboard navigation and screen reader support are prioritized. Can I customize the theme? Absolutely. Override the CSS custom properties in tokens.css to match your brand. Dark and light themes are built in.
